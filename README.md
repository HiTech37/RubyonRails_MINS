# README


## Ruby version

Use ruby 2.7.1 and rails 6.0.3.4

## Installation

```bash
git clone https://github.com/rareone/MINS.git
cd MINS
bundle install --without production
# The bundle may be out of date, install/update it on the instruction instead
rake db:migrate
rails server
```
